# Key Logger – Mini JavaScript Project

This is a simple Key Logger mini project built using HTML, CSS, and JavaScript.
It captures and displays keyboard key press and key release events in real time.

## Features
- Turn On / Turn Off key logging
- Detects key press and key release
- Displays real-time keyboard events
- Clears output when turned off

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
key-logger/
├── index.html
├── style.css
└── script.js

## How to Run
1. Clone or download the project
2. Open `index.html` in any browser
3. Click **Turn On**
4. Press any key
5. Click **Turn Off** to stop logging

## Learning Outcome
- JavaScript keyboard events
- Event listeners
- DOM manipulation
